Tablas cebreadas con jQuery

Esto es una demo de cómo es posible, con jQuery, cebrear y dar un poco de gracia a las tablas de datos en muy poquitas líneas de código. Es complemento de la Introducción a jQuery publicada en Dizque.

Este documento XHTML incluye un pequeño script que se encarga de añadir clases a las filas pares de la tabla, lo que nos permite asignarles estilos vía CSS. Además, se añaden manejadores que manipular el class de cada fila cuando el mouse entra/sale y al hacer clic.

Echa un vistazo al código fuente para ver cómo demonios funciona.
